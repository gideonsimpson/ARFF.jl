ARFF.optimal_γ — Function
optimal_γ(d::Integer)

Compute the optimal γ parameter as a function of dimension d

Fields

  • d - the dimension of the x coordinate
source

Following Remark 1 in [1], the optimal $\gamma$ corresponds to $\gamma = 3d -2$, which is encoded in the above function.
